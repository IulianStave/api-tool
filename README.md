# api-tool
Clockify API tool - copies or deletes time entries

This script can be imported as module


The default configuration JSON file is `local.config.json`
and it should follow the `config.json` structure.

In case you have another configuration file,
it can be passed as command line argument, for instance:

`python3 api.py -j myconfig.json`
or
`python3 api.py --json myconfig.json`


To see the complete list of arguments, run:
`python3 api.py -h`
